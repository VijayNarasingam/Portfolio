Toy Store
https://vijaynarasingam.github.io/yugan_photography/

As a developer for the Toy Store website, I designed and implemented a user-friendly platform focused on delivering a seamless online shopping experience. The website allows customers to browse an extensive catalog of toys, place orders, and track their purchases. By integrating user login and logout features, I ensured that users can securely manage their personal information, view their order history, and easily check out when making new purchases. The frontend is built with clean, responsive HTML and CSS to ensure compatibility across devices, providing an intuitive experience whether on desktop or mobile.

In addition to standard e-commerce functionality, the Toy Store website features a custom toy-making option, which sets it apart from other online retailers. This tool allows users to create their own unique toys by selecting various customization options such as color, material, size, and style. Using JavaScript, I developed an interactive interface where users can see a real-time preview of their custom design. This feature not only enhances user engagement but also provides a fun and personalized experience for customers.

Security and usability are top priorities for this project. I implemented a robust login/logout system using JavaScript to ensure that user data is handled securely, with encrypted passwords and safe session management. The website also includes interactive elements such as the toy customization tool, dynamic product listings, and a simple checkout flow. With HTML, CSS, and JavaScript working in harmony, the Toy Store website provides a modern, secure, and enjoyable shopping experience that encourages customers to return for future purchases and toy creations.
